# Favicon Assets

This folder contains favicon files for the DrugGuard AI application.

## Current Files
- `favicon.svg` - SVG favicon based on the Activity icon (pulse/heartbeat)

## To Add Your Own Favicon

1. Place your SVG file in this folder
2. Update `frontend/index.html` to reference your file:
   ```html
   <link rel="icon" type="image/svg+xml" href="/favicon/your-file.svg" />
   ```

## Supported Formats

- SVG (recommended for modern browsers)
- ICO (for older browser compatibility)
- PNG (for fallback)

You can also generate multiple sizes and formats using tools like:
- https://realfavicongenerator.net/
- https://favicon.io/
